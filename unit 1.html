Software

Software consists of a set of instructions, data, or computer programs that are used to run systems and carry out particular tasks. Programming code that is run on a computer processor is sometimes referred to as computer software. The code may be written at the operating system or machine level.

Examples

  • MacOS
  • Google Chrome
  • Safari
  • Windows

    • Hardware

    The physical parts that make up a computer are referred to as hardware. Each component of a computer that we can physically touch is referred to as hardware. These are the main electronic components that go into making a computer.

    Examples

    • RAM
    • Hard Drive
    • Monitor
    • CPU

    RAM

    Random Access Memory (RAM), is the processor's temporary memory, where information is kept as needed. Contrast this with persistent data, which is kept on your hard drive and is accessible even when your computer is off.


    Hard Drive

    A computer's operating system, software, and data files, including documents, images, and music, are all stored on a hard drive, sometimes known as a hard disc or HDD.


    Monitor

    The video and graphics data produced by a connected computer through the video card is displayed on a monitor, a component of computer hardware. TVs and monitors both display information, but monitors often do so with far greater quality.


    CPU

    Central Processing Unit (CPU), is the central component that serves as a computer's "control center." The CPU, sometimes known as the "central" or "main" processor, is a sophisticated collection of electronic circuitries that manages the device's software and operating system.
